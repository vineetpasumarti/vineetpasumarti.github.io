Autonomous VIO-Based Quadrotor Navigation and Control Stack

MEAM 6200: Advanced Robotics | Spring 2025

Team: Vineet Pasumarti, Ethan Senatore, Allen Zhou, Raymond Yang

Part 1: Crazyflie Lab Report (PDF) Part 2: VIO Implementation Report (PDF)

Project Overview

This project develops a complete autonomy stack for quadrotor navigation that operates without external positioning systems. The core innovation lies in replacing traditional VICON motion capture dependency with an on-board Visual Inertial Odometry (VIO) system using stereo vision and IMU sensor fusion through an Error State Extended Kalman Filter (ES-EKF). This advancement enables true autonomous navigation using only on-board sensors, making the system deployable in GPS-denied and infrastructure-free environments.

The technical implementation addresses the fundamental challenge of drift in IMU-based state estimation, where position errors grow quadratically with time due to double integration of accelerometer data. By fusing stereo vision measurements with inertial data, our ES-EKF optimally combines multiple state estimates, tracking features across frames to predict transformations while using disparity measurements for absolute depth estimation. We discovered that artificially scaling the IMU process noise covariance by a factor of 9 significantly improves performance by appropriately weighting visual measurements over potentially drifting inertial predictions.

Building on our initial VICON-based implementation, the system incorporates A* path planning with Ramer-Douglas-Peucker waypoint sparsification and generates minimum-jerk trajectories for smooth flight. The geometric nonlinear controller gains were increased by approximately 1.2x compared to the VICON version to compensate for the VIO state estimation characteristics. For the extra credit challenge, we implemented an online replanning pipeline with a 7.5-meter planning horizon and 5-meter sensor range, enabling real-time obstacle avoidance and navigation in unknown environments. While the online replanner shows 15-25% longer completion times compared to global planning, it successfully demonstrates the feasibility of exploration in unmapped spaces.

Key Technical Contributions

  • Visual Inertial Odometry: Stereo vision + IMU fusion using Error State Extended Kalman Filter for drift-free state estimation
  • Adaptive Sensor Weighting: Optimized Kalman gain through strategic IMU noise scaling (9x) to prioritize visual measurements
  • Trajectory Generation: Piecewise polynomial minimum-jerk trajectories with 0.2m voxel resolution and adaptive margins
  • Online Replanning: Real-time obstacle avoidance with 7.5m planning horizon and collision prediction at 50Hz
  • Controller Tuning: Geometric nonlinear PD controller with gains optimized for VIO-based state estimation
  • Full Autonomy: Complete independence from external infrastructure - deployable in any environment

  • Online replanner handles dynamic obstacles with 20-timestep replanning frequency
  • Complete ROS integration for real-time processing and control

Technical Note

This project represents a significant advancement from traditional motion-capture-based quadrotor control. The transition to VIO-based state estimation required careful consideration of sensor fusion dynamics, particularly the balance between visual and inertial measurements. Our approach of scaling IMU process noise proved crucial for achieving reliable autonomous flight, demonstrating that sometimes the solution lies not in complex algorithms but in understanding the fundamental characteristics of sensor fusion.
